JOGUE CLÁSSICOS DIRETO DO NAVEGADOR!

-MANUAL DE AJUDA-

O USO DE UM JOYSTICK É ALTAMENTE RECOMENDADO

🎮 CONTROLES

SETAS → Movimento
Z / X → Botões principais
Enter → Start
Shift → Select
Você pode mudar as configs do controle a qualquer momento clicando no ícone de controle 🎮 dentro do emulador.

💾 SALVAR

Clique no ícone de disquete ←💾 dentro do emulador.
Vá em “Save State” para salvar seu progresso no jogo.
Use “Load State” para carregar um jogo salvo.
AVISO: O save é apagado se limpar os dados do navegador.

▭ TELA CHEIA

Use o botão "Tela Cheia" abaixo do jogo para colocar o jogo em tela cheia. Certifique-se de carregar um jogo primeiro, caso contrário, a função de tela cheia não funcionará.
Para sair do modo de tela cheia, pressione a tecla "Esc" no teclado.

🎮 JOYSTICK

Controles via USB e BLUETHOOTH funcionam automaticamente, basta conectar no PC.

AVISO: O SITE NÃO FUNCIONA NO CELULAR, APENAS EM COMPUTADORES
